About Us

The Scheduler is software designed for assignment, attendance and online course management.

Managing classes is easier and more convenient than ever.

Our team consists of students which mean that we know how challenging it is to manage attendance and assignments from a number of classes. Therefore, we did our best to develop the software that may help to overcome those difficulties. Our app allows users to:

  • Register classes.
  • Set alarms before each class.
  • Get notified before assignment deadlines.
  • Have your online classes links notified before each class.
  • Get detailed attendance reports.

This website serves as a documentation for the Scheduler which gives detailed instruction about the each section of the app. If there is something more you want to know about the app contact the developer here

Class Section

In this section, users can register, update, delete and see a list of classes.

To register a class:

Users need to fill out the registration form which requires them to enter:

1. Course name;
2. Professor's name;
3. Class room number - it is optional to enter.
4. Class day/days - it can be a day or multiple days not including weekends
5. Class Link - it is also optional, but it is highly recommended to register it as it will appear on the notification at the alarm time
and when user clicks on notification button it will redirect to online class meeting.
if the link is not given the button will still appear at alam time, but when it is clicked it will open up the course list and prompts to register the link by updating the class:

If the user wants to register it once registration is complete, they should update the class.
6. Start, end, and alarm time for the registered day/days - classes can be registered from 8 to 23 o'clock.
Once class is registered it will appear in class list and timetable. There will be set weekly alarms which will be triggered at the alarm time every week and it can be cancelled only by deleting the class.

To Update a class:

Users need to press and hold on to the class that they want to update from the course list in the class section of the app. The options menu will appear, from which the "update" option should be chosen.

- Users can not update a class to multiple days.
- if the class overlaps with any other class update will not be proceeded.
- if the class is more than one day it should be deleted and reregistered.

To Delete a class:

Users need to select delete option from iptions menu that's shown above.
- once the class is deleted, the attendance infromation is also removed automatically whereas assignments are not.

Timetable Section

This section of the app demonstrates detailed information about each class. It highlights the classes with different colors to make it easy to distinguish them and these colors are changed every time. Besides, the class that is more than one day of the week will have the same color.
To open the timetable from the menu of the app, just swipe the screen to the right or press the timetable button.

Assignment Section

In this section, users can register, delete and see a list of registered assignments.

To register an assignment:

Users need to fill out the registration form which requires them to:

1. Choose subject which assignment's given;
2. Give Assignment Title;
3. Due date - that can not be set to the past;
4. Due time
5. Alarm date - that can not be set later than due date;
4. Alarm time - that also can not be set to the past;

Once assignment is registered, there will be set the alarm that will ring at alarm date and time.

- when user clicks on notification it will open up the assignments list.
- or it can be swiped just to remove the notification from the panel.

To delete an assignment:

Users should just swipe the assignment that they want to delete to the right or the left. Afterwards, there will appear a small dialog where users can delete or keep the assignment by pressing "yes" or "no" button.

-if assignment is kept even after the alarm, it will be removed automatically after the deadline.

Attendance Section

Detailed demonstration of both general and each class-specific attendance information

The pie chart at the top of this section highlights the general attendance information. In other words, it shows the percentage of total attendance, absence, and tardiness in all classes combined.

Below the pie chart, there is a list of registered classes with related course information. To check the dates of each attendance check, users should just click on the class and there will appear dates and attendance data.

- it may take a little time to load the data when this page is opened.
- this data is collected during attendance check which will appear on the notification panel after 5 minutes each class start time:
- this notification can not be swiped off or cleared.
- it is mandatory to select one of the options.
- you can disable attendance check from settings of your phone by turning off the notification from the Scheduler channel 2. (It will disable attendance for every class)

- it is recommended to keep both of channels on.

Frequently Asked Questions

If you have more questions ask from the developer here

  • Why attendance information is not showing in attenance section?

    It takes some time to load the data. We recommend going back and enter again which will reload the data. We are fixing this issue in the coming updates of the app.

  • Do I need to delete class once the semester is over or it is deleted automatically?

    Users should delete the class once it is over while assignments are removed automatically after its deadline.

  • Is there any ways to turn off the notifications?

    You should find the app from Settings -> Apps-> Scheduler part of your your device.
    - turn off channel 1 for disabling "Start class Alarms" and "Assignment Alarms"
    - turn off channel 2 for "Attendance Checks"
